# ShoppingCartzidan
Tugas membuat Shopping Cart menggunakan Flutter


## Yang Sudah Dibuat

- Add to cart dari product list
- Tampilkan cart beserta quantity
- Update quantity (+ / âˆ’)
- Remove item dari cart
- Total harga otomatis
- Search produk berdasarkan nama
- Filter produk berdasarkan kategori
- Halaman checkout (order summary + form)


## Isi File

Semua kode berada di:

- lib/main.dart

Struktur di dalam file:

- HalamanProduk
Menampilkan daftar produk mobil mainan, dilengkapi fitur search dan filter kategori serta tombol tambah ke keranjang.

- HalamanCart
Menampilkan isi keranjang, dapat menambah/mengurangi quantity, menghapus item, dan menampilkan total harga otomatis.

- HalamanCheckout
Menampilkan ringkasan pesanan serta form data pembeli (nama, alamat, nomor HP).

- Fungsi rupiah()
Digunakan untuk mengubah angka menjadi format mata uang Rupiah (Rp).
